# Excel词库上传指南

## 功能说明

现在您可以通过上传Excel文件来创建包含多个关卡的词库。每个Excel页面（Sheet）将自动转换为一个独立的关卡。

## Excel文件格式要求

### 基本格式
- **文件格式**: `.xlsx` 或 `.xls`
- **词库名称**: 文件名（不含扩展名）
- **关卡名称**: 每个Sheet的名称

### 数据格式
每个Sheet应包含两列数据：
- **A列**: 英语单词
- **B列**: 中文翻译

### 示例结构

```
文件名: 初级英语.xlsx

Sheet1: 基础词汇
| A列 (英语)  | B列 (中文) |
|------------|-----------|
| hello      | 你好      |
| world      | 世界      |
| apple      | 苹果      |

Sheet2: 进阶词汇
| A列 (英语)  | B列 (中文) |
|------------|-----------|
| computer   | 计算机    |
| program    | 程序      |
| database   | 数据库    |
```

## 上传步骤

1. **准备Excel文件**
   - 创建Excel文件
   - 为每个关卡创建一个Sheet
   - 在每个Sheet中填入单词对（A列英语，B列中文）

2. **上传词库**
   - 进入"更多设置"页面
   - 点击"创建新词库"
   - 选择"Excel文件上传"方式
   - 点击"选择Excel文件"按钮
   - 选择准备好的Excel文件

3. **确认上传**
   - 系统会自动解析文件
   - 创建词库（名称为文件名）
   - 为每个Sheet创建对应的关卡
   - 导入所有单词对

## 游戏功能

### 关卡切换
- **刷新词汇**: 从当前关卡随机选取新的单词组合
- **下一关**: 切换到下一个关卡，显示该关卡的单词

### 关卡信息显示
- 游戏界面顶部显示当前词库名称
- 显示当前关卡名称
- 显示关卡进度（例如：1/3）

### 排行榜
- 每个关卡有独立的排行榜
- 记录按关卡分别统计

## 注意事项

1. **空行处理**: 系统会自动跳过空行
2. **数据验证**: 只有同时包含英语和中文的行才会被导入
3. **Sheet数量**: 建议每个词库包含2-10个关卡
4. **单词数量**: 每个关卡建议包含10-50个单词对
