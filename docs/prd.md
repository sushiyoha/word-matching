# 英语单词消消乐游戏需求文档

## 1. 游戏概述
### 1.1 游戏名称
英语单词消消乐

### 1.2 游戏描述
一款基于记忆匹配的英语学习游戏，通过翻牌配对的方式帮助用户记忆英语单词及其翻译。支持多种题库和游戏模式，后续将扩展至更多语言。

## 2. 核心功能
### 2.1 题库系统
- 内置词库：包含参考图片中的游戏术语词汇，后续会增加高考3500词
- 自定义词库：用户可上传Excel表格或编辑个人词库
- 多关卡支持：Excel文件中的多个页面作为同一词库的不同关卡，词库名称为文件名，关卡名称为页面名称

### 2.2 游戏模式
- 基础模式：固定10个英语单词进行游戏
- 加词模式：用户可选择增加单词数量，每次加词增加一行（2个单词）
- 联机匹配竞技模式：3v3组队竞技

### 2.3 词库管理
- 支持Excel表格形式上传词库（A列为语言A，B列为语言B）
- 支持在已有词库中直接修改
- 格式：两列对应，英语-翻译
- 词库删除：用户可删除不需要的词库
- 词库查看：点击词库查看按钮进入新页面，该页面以卡牌形式展示词库内容，每张卡牌显示一个词对（英语单词-翻译），卡牌排列方式与游戏界面相同。长按卡牌可选择该卡牌，选择后可进行编辑和删除操作，支持批量选择和全选功能。支持搜索功能快速定位单词
- 添加词对功能：用户可在现有词库中手动添加新的单词-翻译词对，也可通过上传Excel表格进行批量添加
- 公共词库浏览：用户可查看所有其他用户上传的公开词库，并可选择将感兴趣的词库添加到自己的词库中

### 2.4 关卡功能
- 刷新词汇：从当前页面内重新选取词汇进行游戏
- 下一关：切换到下一页面的词汇，页面显示下一关的词汇和关卡名称
- 关卡列表选择：用户可通过关卡列表界面查看当前词库的所有关卡，并选择任意关卡进行游戏，关卡列表以美观简洁的方式展示关卡名称和进度
- 关卡选择：用户可选择词库中的任意关卡进行游戏

### 2.5 游戏控制
- 返回按钮：在游戏界面的显著位置设置返回按钮，用户可随时点击返回主页
- 重新开始功能：开始游戏后提供重新开始当前游戏的选项
- 页面导航：在所有非首页的页面（包括词库管理、关卡列表、排行榜、设置等界面）的左上角或显著位置添加黑色的←返回按钮，用户可点击返回上一级页面

### 2.6 设置功能
- 齿轮图标⚙：在游戏界面添加齿轮图标，用于访问游戏设置选项

### 2.7 语音功能
- 短文本语音合成：点击单词卡牌时会念出单词的读音
- 支持多语言发音：根据单词语言自动选择对应的语音引擎进行发音
- 发音反馈：点击单词时提供视觉反馈，确认语音播放状态

## 3. 玩法说明
### 3.1 游戏流程
1. 首页展示单词及其翻译，每行显示4张卡牌（单词-翻译-单词-翻译）
2. 点击开始游戏后，所有卡牌翻到背面并打乱顺序
3. 游戏界面提供返回按钮和重新开始按钮
4. 用户点击卡牌寻找匹配的单词和翻译
5. 匹配成功的卡牌消失，直至全部消除完成游戏
6. 游戏过程中点击单词卡牌可听取发音

### 3.2 记录系统
- 记录用户完成游戏的步数
- 记录游戏花费时间
- 记录游戏完成日期
- 展示同一关卡下（相同词库-相同单词数）最快完成用户的时间记录
- 排行榜界面：设置为独立界面，每当用户完成一局游戏后，自动更新排行榜数据，按完成时间排序展示最佳成绩，显示用户的排名和第一名的排名
- 游戏结束界面：游戏完成后显示成绩对比界面，包含第一名的成绩、用户个人最好成绩和本局游戏成绩三项数据，便于用户了解自己的表现和进步情况

## 4. 设计风格
界面美观简洁，采用冷色调和淡色调设计，关卡列表等功能界面同样保持简洁美观的视觉效果
